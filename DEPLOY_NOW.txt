================================================================================
                      IF DEPLOYMENT FAILS
================================================================================

The window will show the error and stay open!

Common errors:

1. "Azure CLI not found"
   → Install from: https://aka.ms/installazurecliwindows

2. "Login failed"
   → Run manually: az login
   → Then run the script again

3. "npm install failed"
   → Run manually: npm install
   → Then run the script again
