# 🚀 Deploy to Azure - Quick Start

## Step 1: Install GitHub CLI

Download: **https://cli.github.com/**

## Step 2: Run This Command

Open Command Prompt in this folder and run:

```cmd
gh auth login
```

Follow the prompts to login to GitHub.

## Step 3: Run the Script

```cmd
add-github-secrets.bat
```

This will automatically add all 12 secrets to GitHub.

## Step 4: Watch Deployment

Go to: **https://github.com/Gowtham-BR2005/chart_application/actions**

Both frontend and backend will deploy automatically!
